# Heritage Web App

User-facing web application for Heritage Platform.

## Features

- Browse content in a feed-style interface
- View performer profiles
- Stream and download content
- Search functionality
- Responsive design (mobile & desktop)

## Setup

```bash
npm install
npm run dev
```

Runs on: http://localhost:5175

## Tech Stack

- React 18
- TypeScript
- Material-UI
- React Router
- Axios
- Vite
